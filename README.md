# xFund

A股上市公司供应链数据智能平台

🔗 **在线体验**: https://jackson906e.github.io/xFund/

## 功能

- 📊 覆盖 5,494 家 A 股上市公司供应链数据
- 🕸️ 交互式供应链知识图谱
- 🔍 数据源可验证（公司公告 + 权威新闻）
- ⚡ Python SDK & RESTful API
